# DestroyIt

A small mobile game developed with React Native as a side project to experiment with mobile app development.

## Technologies Used
- React Native
- JavaScript

## Description
DestroyIt is an experimental mobile game built to improve my React Native skills.  
The game is currently in progress and focuses on basic gameplay mechanics and UI design.
The UI design is inspired by popular mobile game interfaces and adapted for this project.
Currently the project includes only the main menu UI as a prototype. Gameplay logic is under development.

## Status
Work in progress. Features and gameplay will be expanded in future updates.
